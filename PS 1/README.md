## PS 1 - Run Instructions

### Prerequisites
- Java JDK 8+ installed and `java`/`javac` available in PATH.

### Compile
From the PS 1 folder:

```
javac *.java
```

### Run
Open two terminals in the PS 1 folder.

Terminal 1 (start server):

```
java Server
```

Terminal 2 (run client):

```
java Client
```

You should see "Server is ready." in the server terminal and the client will print the results of each remote method call.
